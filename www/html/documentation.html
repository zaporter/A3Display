The Board

This is the doc page for the Stodd A3 LED board web interface. Here you can find information on how to interact with the board and put your own cool stuff up there. If you want to post code to the board, you must talk to one of the creators for the passphrase. Once you have that, you can write code in the box on the home page and submit it. Below are some basic commands that you can run to demo some of the board's functionality. We will be adding more as we go. If you want to get deeper in developing an app or script, jump over to our Github to view the API for interacting with the board.

Writing Text

defaultText(text, text_color=(0,0,0), bg_color=(200,200,200), speed=1, loop=1000, size=14)

This is the simplest interface for writing text to the board. RGB values are defined as a tuple with 3 values 0-255 (e.g. (124, 200, 255)). This function will scroll the given text from right to left across the screen. You need to pass the text color and background color, as well as a speed (refresh rate) a good speed to start is 10.

dropText(image img, string text, string font, int wait_ms, int scroll_no)
flashText(text, textColor1, textColor2, bgColor1, bgColor2, speed=10, loop=1000, loc=(0,1), size=14)

Images

pushImage(file filename)

As long as the file exists in the A3Disp directory on the pi, you can display an image on the board with this function.

pushGIFFile(file filename)

Same goes for this function, except you can display GIF formats.
